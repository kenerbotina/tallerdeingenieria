# EcoScan IA ♻️

Una aplicación web minimalista diseñada para clasificar residuos de forma inteligente utilizando inteligencia artificial en tiempo real. 

El proyecto utiliza el modelo de visión artificial de **Teachable Machine** para identificar si un objeto es orgánico, reciclable o no aprovechable, facilitando una correcta disposición final.

## Características
*   **Escaneo en tiempo real:** Clasificación instantánea mediante la cámara del dispositivo.
*   **Interfaz Minimalista:** Diseño limpio enfocado en la usabilidad y la legibilidad.
*   **Adaptable:** Soporte para cámaras frontales y traseras (optimizado para móviles).
*   **Captura y Registro:** Posibilidad de tomar fotografías de los residuos clasificados y guardarlas.

## Stack Tecnológico
*   **Frontend:** HTML5, CSS3, JavaScript (Vanilla).
*   **Inteligencia Artificial:** TensorFlow.js, Teachable Machine.
*   **Despliegue:** GitHub Pages.
